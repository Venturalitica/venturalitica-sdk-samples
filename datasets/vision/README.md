# Vision Datasets

This repository uses the **FairFace** dataset via streaming from Hugging Face Hub. No large local downloads are required.

- Source: `HuggingFaceM4/FairFace`
- License: CC BY 4.0
- Attributes: race (7), gender (2), age (9)

To try the streaming demo, run the scenario in `scenarios/vision-hf-datasets`.
